Protocol
The rules of the internet
Packet
A peice of data to be sent along the internet.
Header
The part of a packet used to show destination and return adress as well as other information.
Payload
The part of a packet that contains the actual data being sent.
Router
A central communication system of the internet. Acts as a midpoint between computers.
Routing Table
Acts as a map so a router knows where to send the information
