# The TCP/IP model

The model has 4 layers

## Application

Protocols in this layer include:

- World Wide Web (www.)
- Email
- VOIP
- SMS
- IRC

## Transport

Protocols in this layer include:

- TCP
- UDP

## Internet

Protcols in this layer include:

- IP adress

## Link (Network Access)

Protocols in this layer include:

- Wifi
- Bluetooth
- LTE
- Ethernet
